│  │  • Duration: Until manually cleared                            │    │
│  │  • Purpose: Offline access                                     │    │
│  │  • Updated: Every server fetch                                 │    │
│  └────────────────────────────────────────────────────────────────┘    │
│                                ↓                                         │
│  ┌────────────────────────────────────────────────────────────────┐    │
│  │  Level 3: Supabase Server (Source of Truth)                    │    │
│  │  • Duration: Permanent                                         │    │
│  │  • Purpose: Central database                                   │    │
│  │  • Accessed: When online                                       │    │
